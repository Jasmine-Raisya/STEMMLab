## Problem Statement
Following developments in education digitalization and the increasing accessibility to mobile devices, STEMM Lab aims to utilize this shift in order to: introduce alternative ways to deliver both on-site learning & home learning programme via Native Mobile App. STEMM Lab aims to solve dependency on physical labs and allow for; guided immersive learning to be possible at any time and any place using appliances that could be described as common tools and objects. By adopting STEMM Lab, both students and educators could benefit from convenience of a portable lab, as well as independent learning at their own pace.
